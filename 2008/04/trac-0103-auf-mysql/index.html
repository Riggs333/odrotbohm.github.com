Trac 0.10.3 auf MySQL

21 April 2008

Ich habe mir bei meinem Arbeitgeber für ein paar kleinere Projekte ein Trac eingerichtet, das gegen eine MySQL Datenbank läuft. Das ganze macht besonders im Zusammenspiel mit Mylyn großen Spaß. Gestern fiel mir jedoch auf, dass ein paar der vorbereiteten Ticketqueries nicht funktionierten. Angeblich sei der Query falsch.

Nach kurzer Recherche stieß ich auf einige die das gleiche Problem hatten. Es handelt sich wohl um einen known Bug, der mit Release 0.10.4 behoben sein wird. Denjenigen die - wie ich - nicht so einfach auf diese Version upgraden können, sei geraten einfach die einfachen Hochkommata um Vorkommen von $User in den Queries zu entfernen. Dann tut alles wieder wie erwartet.
